# MLOps Experiment Tracker

A custom experiment tracking platform inspired by MLflow and Weights & Biases.

## Features

- Experiment Tracking
- Hyperparameter Logging
- Metric Tracking
- Artifact Management
- Model Registry
- Run Comparison Dashboard
- PostgreSQL Cloud Database
- Flask REST APIs

## Tech Stack

- Python
- Flask
- PostgreSQL (Neon)
- SQLAlchemy
- Chart.js
- GitHub
- Render
